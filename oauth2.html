OAuth2

OAuth2 is required for secure access to gmail for access and mail filtering.

See Quick Run-Through with oauth2.py page[1] for configuration details:
    OAuth2DotPyRunThrough
    https://github.com/google/gmail-oauth2-tools/wiki/OAuth2DotPyRunThrough

Configuration

OAuth2 configuration under Debian Buster 10.13 required for secure access to gmail and mail filtering.

  1. Backup ~/.imapfilter/config.lua if it exists:[1]

    $ pushd ~/.imapfilter
    ~/.imapfilter ~
    $ cp config.lua config.lua-20221102

  2. Verify Python version 2.4 or greater required by oauth2.py script.[1]

    $ python -V
    Python 2.7.16

    Looks good.

  3. "download oauth2.py from the source repository on the web"[1]
    https://raw.githubusercontent.com/google/gmail-oauth2-tools/master/python/oauth2.py

    Downloaded oauth2.py script with:
    $ cd ~/bin
    $ wget https://raw.githubusercontent.com/google/gmail-oauth2-tools/master/python/oauth2.py

  4. Set full privs on the script for myself:
    $ cd ~/bin
    $ chmod 700 oauth2.py
  5. 'Registering An Application

    To use OAuth2, you must have registered your application through the Google APIs Console. Registration is explained in Using OAuth 2.0 to Access Google APIs.

    After registering your application, go to the "API Access" tab and create a Client ID. oauth2.py is designed to work with the "Installed application" application type. After creating a Client ID, the API Access tab should show a Client ID and Client secret. You will need to supply these values to oauth2.py in the next step.'[1]

  6. Configuration complete.
    $ popd

Configuration

  1. Backup ~/.imapfilter/config.lua if it exists:

    $ pushd ~/.imapfilter
    ~/.imapfilter ~
    $ cp config.lua config.lua-20220103

  2. Create ~/.imapfilter directory to hold configuration file:

    $ mkdir -p ~/.imapfilter

  3. Create ~/.imapfilter/config.lua configuration file for filtering rules:

    $ touch ~/.imapfilter/config.lua

  4. Add basic configuration to ~/.imapfilter/config.lua:
    options.timeout = 1200
    options.create = true
    options.subscribe = true
    options.expunge = true
  5. Define imap account in ~/.imapfilter/config.lua:
    ~/.imapfilter/config.lua account1 = IMAP {
      server = "imap.gmail.com",
      username = "mylogin@domain.com",
      password = "mypassword",
      ssl = "tls1"
    }
